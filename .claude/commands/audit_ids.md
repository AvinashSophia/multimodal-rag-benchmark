Always ensure image IDs and text chunk IDs remain consistent and relevant across all pipeline stages — dataset loading, retrieval index construction, model generation, and evaluation. Metric computation (Recall@k, MRR, nDCG, Attribution Accuracy, etc.) depends on matching retrieved IDs against ground-truth IDs. Inconsistent IDs silently corrupt metrics. When modifying `UnifiedSample`, `RetrievalResult`, or `Evaluator` code, verify ID propagation end-to-end.
